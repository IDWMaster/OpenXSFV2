Welcome to GlobalGrid web client

This software allows you to connect to GlobalGrid services via a standard web browser.
Please note that although this software is designed with the strictest security in mind,
that malicious software running on your computer could be able to intercept or manipulate this program
in unauthorized manners. For this reason; we recommend using this on a computer that is NOT shared with other users, and does not provide any kind of server services to the outside world, aside from GlobalGrid services.
